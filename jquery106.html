.attr()

선택한 HTML 요소의 속성 값을 설정하거나 반환합니다.

문법(Syntax)

$("선택자").attr("속성");
$("선택자").attr("속성", "속성값");
$("선택자").attr({"속성":"속성값", "속성":"속성값", .... });
$("선택자").attr("속성", 함수);

정의(Definition)

  • .attr() 메서드는 선택한 요소의 속성 값을 가져옵니다.
  • .attr() 메서드는 선택한 요소의 속성 값을 변경합니다.
  • .attr() 메서드는 선택한 요소의 여러개 속성 값을 변경합니다.
  • .attr() 메서드는 선택한 요소의 속성과 함수를 설정합니다.
