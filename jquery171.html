.slideDown()

선택한 요소를 슬라이딩으로 보여줍니다.

문법(Syntax)

$("선택자").slideDown();
$("선택자").slideDown("시간");
$("선택자").slideDown("시간", 함수);

정의(Definition)

  • .slideDown() 메서드는 선택한 요소를 슬라이딩으로 보여줍니다.
  • .slideDown("시간") 메서드는 선택한 요소를 시간동안 슬라이딩으로 보여줍니다
  • .slideDown("시간", 함수) 메서드는 선택한 요소를 슬라이딩으로 보여주고 함수를 설정합니다.

Sample1

.fadeOut() 메서드를 이용하여 요소를 천천히 숨기는 예제입니다.

결과

속성(Attributes)

  • .closest() 메서드는 선택한 요소의 상위 요소 중 가장 가까운 요소를 선택합니다.
  • .find() 메서드는 선택한 요소에서 조건에 맞는 요소를 다시 선택합니다.
  • .next() 메서드는 선택한 요소의 다음 요소를 선택합니다.
  • .nextAll() 메서드는 선택한 요소의 다음 요소를 선택합니다.
리셋 클릭하면 리스트를 .slideDown() 슬라이딩으로 보여줍니다. 클릭하면 리스트를 .slideDown("slow") 슬라이딩으로 보여줍니다. 클릭하면 리스트를 .slideDown(2000) 슬라이딩으로 보여줍니다.
jquery
$(".choice1 .btn1").on("click", function(e){
    $(".list1").slideDown();
});
$(".choice1 .btn2").on("click", function(e){
    $(".list1").slideDown("slow");
});
$(".choice1 .btn3").on("click", function(e){
    $(".list1").slideDown(2000);
});
