.height()

선택한 요소에 높이 값을 설정합니다.

.height()

.height() 메서드는 선택한 요소의 높이(패딩/마진/보더 불포함)를 설정합니다.

문법(Syntax)

$("선택자").height();
$("선택자").height("속성");
$("선택자").height(함수);

정의(Definition)

  • 박스 요소의 높이 값은 패딩, 마진, 보더 값을 포함한 값이기 때문에 .height()는 패딩, 마진, 보더 값을 포함하지 않는 값을 가져옵니다.
  • .height() 메서드와 .css() 메서드는 높이 값을 똑같이 가져옵니다. 하지만 .css() 메서드는 단위까지 출력이 된다는 차이점이 있습니다. 수학적인 계산이 필요한 부분에서는 .hieight() 메서드를 추천하며, 샘플 1번을 참고하면 됩니다.
  • .height() 메서드는 선택한 요소의 높이 값을 가져옵니다.
  • .height("속성") 메서드는 선택한 요소의 높이 값을 변경합니다.
  • .height(함수) 메서드는 선택한 요소의 함수를 이용하여 높이 값을 설정합니다.

.height() : 원하는 요소의 높이 값을 출력하는 예제입니다.

Sample1

탐색(Traversing)

  • .eq() 메서드는 선택한 요소에 인덱스 번호에 해당하는 요소를 찾습니다.
  • .filter() 메서드는 선택된 요소에서 선택하거나 함수를 사용할 수 있습니다.
  • .first() 메서드는 선택된 요소에서 첫번째 자식 요소를 찾습니다.
  • .has() 메서드는 선택된 요소의 자식 요소에서 주어진 선택자를 찾습니다.
  • .is() 메서드는 선택된 요소 객체의 특징을 판별합니다.
  • .last() 메서드는 선택된 요소에서 마지막 번째 자식 요소를 찾습니다.

리셋 클릭하면 리스트의 높이 값(.height())을 구합니다. 클릭하면 브라우저의 높이 값(.css("height"))을 구합니다. 클릭하면 이 문서의 높이 값을 구합니다. 클릭하면 브라우저의 높이 값을 구합니다.
jquery
function showHeight(element, height){
    $(".list1 p").text( element + "의 높이 값은 " + height + "px.");
}
$( ".choice1 .btn1" ).click(function(){
    showHeight("리스트(.height())", $(".list1").height());
});
$( ".choice1 .btn2" ).click(function(){
    showHeight("리스트(.css())", $(".list1").css("height"));
});
$( ".choice1 .btn3" ).click(function(){
    showHeight("문서", $(document).height());
});
$( ".choice1 .btn4" ).click(function(){
    showHeight("브라우저", $(window).height());
});
