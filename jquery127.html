.position()

선택한 요소의 위치 좌표 값(기준점 기준)을 설정하거나 반환합니다.

.position()

선택한 요소의 위치 좌표 값(기준점 기준)을 설정하거나 반환합니다.

문법(Syntax)

$("선택자").position();

정의(Definition)

  • .position() 메서드는 선택한 요소에 위치 좌표 값을 가져옵니다.
  • .position() 메서드의 기준 점은 CSS에서 position:relative에 해당합니다.

Sample3

원을 드래그 하면 원의 좌표값을 표현합니다.
jquery

    drag:function(){
        $('.desc .ol').text( $('.circle').offset().left + "px");
        $('.desc .ot').text( $('.circle').offset().top + "px");
        $('.desc .pl').text( $('.circle').position().left + "px");
        $('.desc .pt').text( $('.circle').position().top + "px");
    }
});
