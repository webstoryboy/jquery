.prop()

선택한 JavaScript 요소의 속성 값을 설정하거나 반환합니다.

.prop()

정의
  • .prop() 메서드는 선택한 요소의 속성 값을 가져옵니다.
  • .prop() 메서드는 선택한 요소의 속성 값을 변경합니다.
  • .prop() 메서드는 선택한 요소의 여러개 속성 값을 변경합니다.
  • .prop() 메서드는 선택한 요소의 속성과 함수를 설정합니다.
문법

$("선택자").prop("속성");
$("선택자").prop("속성", "속성값");
$("선택자").prop({"속성":"속성값", "속성":"속성값", .... });
$("선택자").prop("속성", 함수);
