.innerWidth()

선택한 요소의 가로(패딩 포함, 보더/마진 불포함)를 설정합니다.

.innerWidth()

.innerWidth() 메서드는 선택한 요소의 가로(패딩 포함, 보더/마진 불포함)를 설정합니다.

문법(Syntax)

$("선택자").innerWidth();
$("선택자").innerWidth("속성");
$("선택자").innerWidth(함수);

정의(Definition)

  • 박스 요소의 가로 값은 패딩, 마진, 보더 값을 포함한 값이기 때문에 .innerWidth()는 마진과 보더 값을 포함하지 않는 값을 가져옵니다.
  • .innerWidth() 메서드는 선택한 요소의 가로 값을 가져옵니다.
  • .innerWidth("속성") 메서드는 선택한 요소의 가로 값을 변경합니다.
  • .innerWidth(함수) 메서드는 선택한 요소의 함수를 이용하여 가로 값을 설정합니다.

샘플(Sample)

.innerWidth() : 원하는 요소의 가로 값을 출력하는 예제입니다.

탐색(Traversing)

  • .eq() 메서드는 선택한 요소에 인덱스 번호에 해당하는 요소를 찾습니다.
  • .filter() 메서드는 선택된 요소에서 선택하거나 함수를 사용할 수 있습니다.
  • .first() 메서드는 선택된 요소에서 첫번째 자식 요소를 찾습니다.
  • .has() 메서드는 선택된 요소의 자식 요소에서 주어진 선택자를 찾습니다.
  • .is() 메서드는 선택된 요소 객체의 특징을 판별합니다.
  • .last() 메서드는 선택된 요소에서 마지막 번째 자식 요소를 찾습니다.

리셋 클릭하면 리스트의 가로 값(.width())을 구합니다. 클릭하면 리스트의 가로 값(.innerWidth())을 구합니다. 클릭하면 리스트의 가로 값(.outerWidth())을 구합니다.
jquery
function showHeight(element, height){
    $(".list1 .bLine").text( element + "의 가로 값은 " + height + "px.");
}
$( ".choice1 .btn1" ).click(function(){
    showHeight("리스트(.width())", $(".list1 ul").width());
});
$( ".choice1 .btn2" ).click(function(){
    showHeight("리스트(.innerWidth())", $(".list1 ul").innerWidth());
});
$( ".choice1 .btn3" ).click(function(){
    showHeight("리스트(.outerWidth())", $(".list1 ul").outerWidth());
});
