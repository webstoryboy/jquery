.css()

선택한 요소에 CSS 속성 값을 설정하거나 변경합니다.

.css()

정의
  • .css() 메서드는 선택한 요소의 속성값을 가져옵니다.
  • .css() 메서드는 선택한 요소의 속성값을 변경합니다.
  • .css() 메서드는 선택한 요소의 여러개 속성 값을 변경합니다.
  • .css() 메서드는 선택한 요소의 함수를 설정합니다.

문법

$("선택자").css("속성");
$("선택자").css("속성","속성값");
$("선택자").css({"속성":"속성값", "속성":"속성값", .... });
$("선택자").css("함수");
