.css()

선택한 요소에 CSS 속성 값을 설정합니다.

.css()

.css() 메서드는 선택한 요소에 클래스 존재 여부에 따라 효과를 추가하거나 제거하는 메서드입니다.

문법(Syntax)

$("선택자").css("속성");
$("선택자").css(["속성","속성",...]);
$("선택자").css("속성","속성값");
$("선택자").css("속성",함수);

정의(Definition)

  • .css("속성") 메서드는 선택한 요소에 속성 값을 가져옵니다.
  • .css(["속성","속성",...]) 메서드는 선택한 요소에 여러가지 속성 값을 배열을 통해 가져옵니다.
  • .css("속성","속성 값") 메서드는 선택한 요소에 속성 값을 변경합니다.
  • .css("속성",함수) 메서드는 선택한 요소에 함수를 이용하여 속성 값을 설정합니다.

샘플(Sample)

.css(propertyName) : 선택한 요소에 속성 값을 알아내는 예제입니다.

Sample1

탐색(Traversing)

  • .eq() 메서드는 선택한 요소에 인덱스 번호에 해당하는 요소를 찾습니다.
  • .filter() 메서드는 선택된 요소에서 선택하거나 함수를 사용할 수 있습니다.
  • .first() 메서드는 선택된 요소에서 첫번째 자식 요소를 찾습니다.
  • .has() 메서드는 선택된 요소의 자식 요소에서 주어진 선택자를 찾습니다.
  • .is() 메서드는 선택된 요소 객체의 특징을 판별합니다.
  • .last() 메서드는 선택된 요소에서 마지막 번째 자식 요소를 찾습니다.

            <li>.last() 메서드는 선택된 요소에서 마지막 번째 자식 요소를 찾습니다.</li>
